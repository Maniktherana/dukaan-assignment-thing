## Setup Instructions

 1. Clone repository:
    ```bash
    git clone https://github.com/Maniktherana/dukaan-assignment-thing.git
    ```
 2. Install
    ```bash
    pnpm install
    ```
 3. Run
    ```bash
    pnpm run dev
    ```
